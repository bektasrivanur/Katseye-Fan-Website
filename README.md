# KATSEYE Fan Website

This project is a fan-made website created to introduce the global girl group **KATSEYE**. The website is designed to present clear information, a clean layout, and an easy navigation structure for users who want to learn more about the group.

## About the Project

The website focuses on introducing KATSEYE as a group, highlighting their identity, music, and achievements. Each section is separated into individual pages to keep the content organized and user-friendly. It's created for learning and practice purposes and 

## Features

* **About Section**
  Introduces who KATSEYE is, their background, formation process, and global concept.

* **Members Page**
  Detailed introductions of all six members, including personal information and profiles.

* **Albums Section**
  Information about KATSEYE’s albums, including *SIS (Soft Is Strong)* and *Beautiful Chaos*, with brief descriptions of each release.

* **Songs / Discography Page**
  A page dedicated to song introductions and the group’s musical releases.

* **Market Page**
  A simple market-style section featuring items and a "Buy Now" button that redirects users to external links.

## Technologies Used

* HTML5
* CSS3

## Project Purpose

This project was created for learning and practice purposes. It aims to improve skills in HTML and CSS, layout structuring, and basic UI design while building a complete multi-page website.
